## 1. 产品概述

传统宣纸生产管理系统，面向宣纸制造企业的全流程数字化管理平台，涵盖从原料入库到成品溯源的完整生产链条，助力宣纸厂实现规范化、可追溯的传统工艺管理。

- 解决宣纸生产工序复杂、原料配比依赖经验、质量检验缺乏标准化、批次溯源困难等痛点
- 目标用户：宣纸厂管理层、车间操作人员、质检人员、销售与仓储人员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 厂长/管理员 | 系统分配 | 全部功能访问、数据统计、工艺参数配置 |
| 车间操作员 | 管理员创建 | 原料登记、制浆记录、捞纸登记、晒纸操作 |
| 质检员 | 管理员创建 | 成纸检验、规格分级、检验报告生成 |
| 销售人员 | 管理员创建 | 订单管理、销售台账、溯源查询 |

### 2.2 功能模块

1. **原料管理**：青檀皮/稻草原料入库登记、库存管理、原料质量评估、供应商管理
2. **制浆工序**：燎草制浆记录、配浆比例管理、制浆过程监控、浆料质检
3. **捞纸生产**：捞纸帘床管理、捞纸操作记录、纸坯登记、产量统计
4. **晒纸焙干**：火墙晒纸管理、焙干温度监控、晾晒排期、干燥质量记录
5. **检验分级**：成纸检验标准、规格分级管理、检验报告、不合格品处理
6. **订单生产**：书画用纸订单、文房四宝供应、生产排期、订单跟踪
7. **销售溯源**：批次溯源码、销售台账、工艺传承档案、质量追溯

## 3. 核心流程

用户从原料入库开始，经过燎草制浆、配浆、捞纸、晒纸焙干、检验分级，最终形成成品入库并关联订单销售，全程可溯源。

```mermaid
flowchart TD
    "原料入库（青檀皮/稻草）" --> "燎草制浆"
    "燎草制浆" --> "配浆（皮草比例）"
    "配浆（皮草比例）" --> "捞纸（帘床操作）"
    "捞纸（帘床操作）" --> "晒纸焙干（火墙）"
    "晒纸焙干（火墙）" --> "成纸检验"
    "成纸检验" --> "规格分级"
    "规格分级" --> "成品入库"
    "成品入库" --> "订单匹配"
    "订单匹配" --> "销售出库"
    "销售出库" --> "批次溯源"
    "检验不合格" --> "返工/报废"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：宣纸米白（#F5F0E8）为底色，古铜赭石（#8B4513）为强调色，墨色（#2C2C2C）为文字色
- **辅助色**：靛蓝（#4A6FA5）用于操作按钮、数据图表；苔绿（#6B8E6B）用于成功/完成状态
- **风格方向**：东方文韵·雅致工坊 —— 融合传统水墨意境与现代工业管理界面，以宣纸肌理为视觉基底，搭配毛笔笔触装饰元素与简洁的数据面板
- **字体**：标题使用思源宋体（Noto Serif SC），正文使用思源黑体（Noto Sans SC）
- **按钮风格**：圆角矩形（8px），赭石色主按钮带微弱宣纸纹理，靛蓝次要按钮
- **布局**：左侧固定导航栏 + 顶部面包屑 + 右侧内容区，卡片式模块布局
- **装饰元素**：页面角落的水墨晕染效果、表格标题旁的毛笔笔触装饰线、页面切换时的宣纸翻页动画
